## About The Project

`SystemAudioTranscriber` is a frontend-only web application that provides real-time audio transcription. It's built with modern web technologies and is designed to be deployed to static hosting platforms like Vercel or Netlify.

---

## Features

*   **Real-Time Audio Capture:** Captures audio from your microphone.
*   **Multi-Provider AI Transcription:** Supports a variety of AI providers for speech-to-text conversion.
*   **User-Provided API Keys:** Users can enter their own API keys for transcription services.
*   **Modern UI:** A clean, responsive, and user-friendly interface.

## Getting Started

### Prerequisites

*   [Node.js](https://nodejs.org/en/) (v22.x or later)
*   [npm](https://www.npmjs.com/)

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/chirag127/SystemAudioTranscriber-RealTime-SystemAudio-To-Text-Windows-App.git
    cd SystemAudioTranscriber-RealTime-SystemAudio-To-Text-Windows-App
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

---

## Usage

1.  **Start the development server:**
    ```bash
    npm run dev
    ```

2.  Open your browser to `http://localhost:5173`.
3.  Enter your API key for the desired transcription service.
4.  Click the "Start" button to begin transcription.
